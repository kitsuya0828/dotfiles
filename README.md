# dotfiles

This is [kitsuya0828](https://github.com/kitsuya0828)'s dotfiles.

## Installation

```bash
cd ~
git clone git@github.com:kitsuya0828/dotfiles.git
cd dotfiles
```

### Vim

```bash
chmod +x ./setup-vim.sh
./install.sh
```

### Neovim

```bash
chmod +x ./setup-nvim.sh
./setup-nvim.sh
source ~/.bashrc
```
